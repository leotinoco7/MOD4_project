# MOD4_project

API - Games

## Clonando

````bash
git clone https://github.com/leotinoco7/MOD4_project.git
````

### Installation

```bash
$ npm install
````

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## 🛠️ Construído com

- NodeJS | NestJs | TypeScript
